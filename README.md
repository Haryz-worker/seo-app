# onpage-seo-app

A minimal 3-stage on-page SEO pipeline:

- **Stage A (Fetcher)**: downloads HTML in-memory.
- **Stage B (Extractor)**: converts HTML -> clean JSON (meta, headings, text, links, images).
- **Stage C (Analyzer)**: computes pixel-length for title/meta, readability, links, and focus keyword stats.
